# Project Summary
The Call Kaids Roofing project is an AI-driven digital platform tailored for the roofing industry, aimed at enhancing operational efficiency and optimizing marketing strategies. The platform automates job scheduling, generates targeted marketing content, and improves customer engagement through advanced AI technologies, ultimately driving business growth and ensuring data accuracy.

# Project Module Description
The project consists of several key functional modules:
1. **Internal Operations AI Module**: Automates job scheduling, provides operational insights, and synchronizes documents.
2. **Content Engine**: Generates SEO-optimized blog content, case studies, and testimonials using internal data.
3. **Meta Ads Engine**: Manages advertising campaigns on Meta platforms, utilizing AI for creative generation and audience targeting.

# File Description Inventory
- **public/knowledge-base/**: Organized knowledge files categorized into system governance, brand voice, operations SOPs, marketing content, workflows, data databases, and schemas.
- **src/components/**: Reusable UI components such as buttons, cards, and modals.
- **src/pages/**: Main application pages, including the AI Digital Engine and internal operations.
- **src/lib/**: Utility libraries for handling knowledge base parsing and data loading.
- **src/App.tsx**: Main application routing file, configured to redirect the root to `/internal/v2/home` and maintain a comprehensive routing structure.

# Technology Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS
- **Backend**: Supabase, Google Drive API, Weather API, Meta Ads API
- **AI Tools**: OpenAI, Pinecone, LangChain

# Usage
1. **Install Dependencies**: 
   - Run `pnpm install` to install all required packages.
2. **Build the Project**: 
   - Use `pnpm run build` to compile the project.
3. **Run the Project**: 
   - Execute `pnpm run start` to launch the application.
