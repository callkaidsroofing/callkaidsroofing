# MKF Source Files

This directory contains the organized Master Knowledge Framework files extracted from `knowledge-base-organized.zip`.

## Structure

```
source/
├── 01_business_core/       # Business identity, contact info, brand guidelines
├── 02_services_pricing/    # Service offerings, pricing models, SOPs
├── 03_operations/          # Operational procedures, safety protocols
├── 04_geographic/          # Service areas, suburb coverage
├── 05_marketing_sales/     # Marketing copy, sales templates
├── 06_technical/           # Technical specifications, materials
└── 07_ai_workflows/        # AI workflow definitions (GWA files)
```

## Files to Extract

Extract all 38 files from `knowledge-base-organized.zip` into this directory structure.

Refer to `MASTER_INDEX.json` for file relationships and metadata.

## Usage

These files are synced to the `knowledge_files` database table and embedded into the RAG system for AI-powered context retrieval.
