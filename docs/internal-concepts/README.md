# Internal System Concepts Archive

This folder preserves the ideas from the removed internal/admin website surface.
It is intentionally not a source-code archive.

Use these notes when resurfacing the concepts in another project:

- `inspection-quote-workflow.md` captures the field inspection, scope, quote, and export flow.
- `crm-operations-model.md` captures lead, quote, job, KPI, and dashboard concepts.
- `content-knowledge-system.md` captures content, media, SEO, knowledge-base, and document-management concepts.
- `implementation-boundaries.md` captures what should not be copied from the removed website implementation.

Do not treat these files as production requirements without revalidating current business facts, warranty terms, pricing, integrations, and privacy obligations.
