# Call Kaids Roofing - Alternative Color Palettes

Based on logo analysis:
- **Primary Logo Colors**: Silver/Metallic White (#E8E8E8), Deep Black (#1A1A1A), Bright Blue (#00A8FF - #00C3FF)
- **Logo Style**: Bold, professional, high-contrast with metallic shimmer effect
- **Slogan Blue**: Electric blue (#3399FF range) used in "Proof In Every Roof"

---

## Palette 1: **Professional Steel & Electric Blue** (Recommended)
*Matches logo's metallic silver and bright blue*

```css
--primary: 199 100% 50%;           /* #00A8FF - Electric Blue (logo phone) */
--primary-foreground: 0 0% 100%;
--primary-dark: 199 100% 38%;      /* #006FCC - Deeper blue */
--primary-glow: 199 100% 65%;      /* #52C4FF - Light glow */

--secondary: 0 0% 10%;             /* #1A1A1A - Deep Black (logo bg) */
--secondary-foreground: 0 0% 91%;  /* #E8E8E8 - Metallic silver */

--accent: 0 0% 75%;                /* #BFBFBF - Steel gray */
--accent-foreground: 0 0% 10%;

--charcoal: 0 0% 10%;              /* #1A1A1A */
--steel: 0 0% 75%;                 /* #BFBFBF - Brushed metal */
--silver: 0 0% 91%;                /* #E8E8E8 - Logo text color */
```

**Vibe**: Bold, industrial, premium. Emphasizes the metallic craftsmanship feel.

---

## Palette 2: **Sky Blue & Charcoal Depth**
*Lighter, more approachable while keeping logo DNA*

```css
--primary: 199 95% 55%;            /* #1AB2FF - Brighter sky blue */
--primary-foreground: 0 0% 100%;
--primary-dark: 207 80% 35%;       /* #0F5799 - Ocean depth */
--primary-glow: 199 90% 70%;       /* #66CCFF - Soft glow */

--secondary: 210 30% 18%;          /* #20252E - Softer charcoal */
--secondary-foreground: 0 0% 95%;

--accent: 45 100% 60%;             /* #FFDB4D - Gold accent (premium feel) */
--accent-foreground: 0 0% 10%;

--charcoal: 210 30% 18%;
--gold: 45 100% 60%;               /* Premium highlight */
--cloud: 210 20% 92%;              /* #E6E9ED - Soft white */
```

**Vibe**: Trustworthy, approachable, premium without being harsh. Gold adds warmth.

---

## Palette 3: **Navy Authority & Cyan Energy**
*Professional corporate with modern tech accent*

```css
--primary: 207 90% 32%;            /* #0B4A85 - Deep navy blue */
--primary-foreground: 0 0% 100%;
--primary-dark: 207 100% 20%;      /* #003366 - Midnight navy */
--primary-glow: 207 85% 45%;       /* #1466A8 */

--secondary: 0 0% 8%;              /* #141414 - True black */
--secondary-foreground: 0 0% 95%;

--accent: 180 100% 50%;            /* #00FFFF - Cyan (tech/energy) */
--accent-foreground: 0 0% 10%;

--charcoal: 0 0% 8%;
--cyan: 180 100% 50%;              /* Electric accent */
--platinum: 0 0% 88%;              /* #E0E0E0 */
```

**Vibe**: Corporate authority, tech-forward, serious professionalism with energetic accent.

---

## Implementation Notes

Each palette maintains:
- High contrast (logo style)
- Metallic/professional feel
- Blue as core brand color
- Dark foundations (black/charcoal)

**Recommended**: Palette 1 - Most faithful to logo while adding vibrancy.
